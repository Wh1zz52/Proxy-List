# Universal Proxy Lists

A comprehensive collection of free proxy servers for development, testing, and data collection purposes. All proxies are formatted for easy integration into your tools and scripts.

## 📁 Available Proxy Types

This repository contains proxies of the following types:

| Type | File | Format |
|------|------|--------|
| **HTTP** | [`http.txt`](http.txt) | `IP:PORT` |
| **HTTPS** | [`https.txt`](https.txt) | `IP:PORT` |
| **SOCKS4** | [`socks4.txt`](socks4.txt) | `IP:PORT` |
| **SOCKS5** | [`socks5.txt`](socks5.txt) | `IP:PORT` |
